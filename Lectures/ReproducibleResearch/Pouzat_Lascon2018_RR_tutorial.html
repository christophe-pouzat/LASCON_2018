What follows is an adaptation of Mastering Markdown.

2.1 Let's start our first Markdown file!

Open a text file, that you can call my_first_markdown.md or anything else you like, with your favorite text editor and type: 

It's very easy to make some words **bold** and other words *italic* with Markdown. 
You can even [link to Lascon's program!](http://sisne.org/lascon-vii/program/?lang=en)

_This will also be italic_

**This text will be bold**
__This will also be bold__

_You **can** combine them_

2.9 Some Markdown flavors (like the GitHub one) provide many additional features

2.9.1 Syntax highlighting

Section [5.1.3 of Python's tutorial](https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions)
introduces the _list comprehension_ with:

```python
squares = [x**2 for x in range(10)]
```

2.9.2 Tables

You can create tables by assembling a list of words and dividing them with hyphens - (for the first row), and then separating each column with a pipe |: 

First Header | Second Header
------------ | -------------
Content from cell 1 | Content from cell 2
Content in the first column | Content in the second column

Download and install Pandoc!

2.12 Add a comment (or a tag)

To add a comment (or a tag within a comment) use:

3 Finding your way in your files

3.1 Desktop search software

Use a desktop search software like DocFetcher (platform independent) or Tracker (Unix/Linux) or anyone you like.

3.2 Add tags

Add tags especially to files that are not in text format. There is an ISO standard called Extensible Metadata Platform that specifies how one can add meta-data to image and PDF files. Software like:

  • exiftool
  • Exiv2

allow us to access and modify images metadata.
